Privacy Policy

Last updated: March 7, 2026

Overview

GroupMe Calendar Monitor is a private, family-use application that automatically syncs sports and activity schedules from GroupMe group chats to Google Calendar. It is not a commercial product and is not available to the public.

Information Collected

This application accesses:

  • GroupMe messages from configured group chats to detect scheduled events (games, practices, etc.)
  • Google Calendar to create, update, or remove calendar events
  • Phone numbers of the two family members who configured the system, solely for SMS notifications

How Information Is Used

Message content is analyzed solely to detect schedulable events (date, time, location). No messages are stored permanently, shared with third parties, or used for any purpose other than calendar synchronization.

Data Storage

The application stores only:

  • Message IDs (to avoid reprocessing)
  • Google Calendar authentication tokens (for API access)

All data is stored locally on a private device. No data is transmitted to external servers beyond the necessary API calls to GroupMe, Google Calendar, Anthropic (for natural language processing), and Twilio (for SMS delivery).

SMS Notifications

SMS messages are sent only to the two family members who own and operate this application. Recipients consent by configuring their own phone numbers. Messages can be stopped at any time by removing the phone number from the configuration. No marketing or promotional messages are sent.

Third-Party Services

  • GroupMe API - to read group messages
  • Google Calendar API - to manage calendar events
  • Anthropic API - to analyze message content for event detection
  • Twilio - to deliver SMS notifications

Data Sharing

No personal data is sold, shared, or disclosed to any third parties beyond the API services listed above, which are used solely for the application's core functionality.
